Facelets Tags

  • ui:composition
    Defines a page fragment that is included in another facelet page. It can be used to define a template client.
  • ui:define
    Allows you to define content inside a template.
  • ui:include
    Includes content from another Facelets file into the current view.
  • ui:insert
    Used within a template to declare a placeholder that is filled by content defined in the template client.
  • ui:decorate
    Similar to ui:include, but also allows content within its body to override sections defined in the included file
  • ui:remove
    Designates that the content within this tag should not be rendered.
  • ui:repeat
    A data iteration component, similar to h:dataTable but more flexible as it does not render its own HTML table markup.
  • ui:param
    Declares a parameter that can be passed to included files or templates.
  • ui:fragment
    A grouping component that does not produce any HTML output, useful for conditional rendering.
