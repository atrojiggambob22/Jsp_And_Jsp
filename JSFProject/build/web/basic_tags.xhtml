Basic Tags
Basic Tags

  • h:body
    Renders an HTML body tag.
  • h:button
    Renders an HTML button that can cause a navigation response.
  • h:commandButton
    Renders a submit button that can perform actions or send data to the server.
  • h:commandLink
    Renders a link that acts like a form submit button.
  • h:dataTable
    Renders an HTML table for displaying arrays or collections of data.
  • h:form
    Renders an HTML form allowing components within it to submit data.
  • h:graphicImage
    Renders an HTML img tag.
  • h:head
    Renders the head section of the HTML document.
  • h:inputHidden
    Renders an HTML input tag of type "hidden".
  • h:inputSecret
    Renders an HTML input tag for hiding its input (password).
  • h:inputText
    Renders an HTML input tag of type "text".
  • h:inputTextarea
    Renders an HTML textarea element.
  • h:message
    Displays messages for a specific JSF component.
  • h:messages
    Displays messages for multiple JSF components.
  • h:outputFormat
    Renders parameterized text.
  • h:outputLabel
    Renders an HTML label element.
  • h:outputLink
    Renders an HTML anchor tag.
  • h:outputText
    Renders static text.
  • h:panelGrid
    Renders an HTML table to organize components.
  • h:panelGroup
