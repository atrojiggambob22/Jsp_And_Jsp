Core Tags

  • f:ajax
    Adds Ajax behavior to JSF components to execute partial page updates.
  • f:attribute
    Adds an attribute to a JSF component.
  • f:convertDateTime
    Attaches a date/time converter to a JSF component.
  • f:convertNumber
    Attaches a number converter to a JSF component.
  • f:converter
    Generic tag for attaching a custom converter to a JSF component.
  • f:event
    Registers an event listener for a JSF component.
  • f:facet
    Defines a facet within a JSF component.
  • f:metadata
    Contains view-level metadata.
  • f:param
    Defines parameters that can be passed to an action method.
  • f:selectItem
    Represents a single selectable item inside a select component.
  • f:selectItems
    Represents a collection of selectable items.
  • f:setPropertyActionListener
    Sets the value of a bean property when an action is performed.
  • f:subview
    Encapsulates a subset of the view to allow inclusion in another view.
  • f:validateBean
    Bean validation within JSF.
  • f:validateLength
    Validates the length of a value.
  • f:validateLongRange
    Validates whether a value is within a specified long range.
  • f:validateDoubleRange
    Validates whether a value is within a specified double range.
  • f:validateRequired
    Validates that a field has been filled out.
  • f:validator
    Attaches a custom validator to a JSF component.
  • f:valueChangeListener
    Registers a value change listener for a JSF component.
  • f:view
    Represents a JSF view.
  • f:viewAction
    Binds a method that is invoked upon view rendering.
  • f:viewParam
    Declares a parameter that is expected in the view request.
